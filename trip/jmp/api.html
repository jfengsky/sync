Jmp cQuery Plugin Demo Page

Options

group String
浮动层所在的组
默认值为: 'group'
alignTo String
浮动层初始位置的针对对象
可选的值为: 'target','cursor'
followCursor Boole
是否跟随鼠标滚动
可选的值为: 'true','false'
allowTipHover Boole
浮动层是否允许鼠标放上去
可选的值为: true','false'
showTimeout Number
显示时的延时
默认的值为: '500'
hideTimeout Number
隐藏时的延时
默认的值为: '100'
timeOnScreen Number
在屏幕上可持续的事件，0表示无限期
默认的值为: '0'
hideTip Boole
鼠标离开浮动层时是否隐藏浮动层
可选的值为: 'true','false'
showArrow Boole
是否显示箭头
可选的值为: 'true','false'
boundaryShow Boole
指定位置越界后是否依旧显示
可选的值为: 'true','false'
position String
位置
默认的值为: 'bottomLeft-topLeft'
maxWidth Number
最大宽度
默认的值为: '260'
minWidth Number
最小宽度
默认的值为: '50'
type String
浮动层的类型
默认的值为: 'jmp_alert'
可选的值为: 'jmp_alert','jmp_text','jmp_title','jmp_table','jmp_tab_list'
template String
浮动层的模板
默认的值为: '#jmp_alert'
可选的值为: 'jmp_alert','jmp_text','jmp_title','jmp_table','jmp_tab_list'
templateUrl String
浮动层的模板服务器地址
默认的值为: '/code_beta/weixj/cQuery/template/'
dataUrl String
浮动层的数据服务器地址
默认的值为: '/code_beta/weixj/cQuery/data/'
eventType String
默认的事件类型
可选的值为: 'def','input','widget'
events Object
触发事件类型
默认选的值为:
{ def:"mouseover,mouseout",
// default show/hide events for an element
input: "focus,blur",
// for all input elements
widget: "focus|mouseover,blur|mouseout"
// select, checkbox, radio, button
}
content Object
默认的内容
默认选的值为:
{ "txt":
"今天是2010年上半年最后一个交易日，沪深股市在昨日暴跌阴影下继续下探，最终跌破2400点整数关口。统计数据显示，上证综指与深证成指今年来分别下跌26.82%和31.48%。"
}
onBeforeShow Function
显示前的函数，参数为当前浮动层元素
默认的值为: 'function(obj){}'
onBeforeHide Function
显示后的函数，参数为当前浮动层元素
默认的值为: 'function(obj){}'
onShow Function
隐藏前的函数，参数为当前浮动层元素
默认的值为: 'function(obj){}'
onHide Function
隐藏后的函数，参数为当前浮动层元素
默认的值为: 'function(obj){}'
scroll Boole
是否支持滚动条滚动事件
默认的值为: 'false'
可选的值为: 'true','false'
resize Boole
是否支浏览器缩放事件
默认的值为: 'false'
可选的值为: 'true','false'
tip String
浮动层最外层的类
默认的值为: 'tuna_jmpinfo'
box String
浮动层的类
默认的值为: 'base_jmp'
tip String
浮动层类型的类
默认的值为: 'jmp_alert'
可选的值为: 'jmp_alert_t','jmp_alert_b','jmp_alert_l','jmp_alert_r'
boxArrow String
箭头外层的类
默认的值为: 'base_jmp_t'
可选的值为: 'base_jmp_t','base_jmp_b','base_jmp_l','base_jmp_r'
arrow String
浮动层箭头的类
默认的值为: 'tri_t'
可选的值为: 'tri_t','tri_b','tri_l','tri_r'
loading String
浮动层加载时的类 -- 暂未实现
默认的值为: 'JT_loader'
templs Object
浮动层的模板
默认的值为: {
"tip":'< div id=${id} class=${tip} group=${group} style="visibility:hidden;display:block;z-index:99;margin:0;left:-9999px;top:-9999px;overflow:hidden;position:absolute;max-width:${maxWidth}px;min-width:${minWidth}px;_width:expression((this.offsetWidth>${maxWidth})?\'${maxWidth}px\':true);"> < div class="${box} ${boxType} ${boxArrow}"> <b class="${arrow}">< /b> < div class="${loading}"> < /div> ${content}< /div> < /div >' }

Methods

getTip Function
获取页面元素对应的控件实例
/**
*获取页面元素对应的控件实例
* @param {Object} obj cQuery对象
* @return {Object|null} oJmp cQuery对象对应的jmp控件的实例
*/
getTip($("id"))
show Function
执行该元素绑定的所有显示函数
show()
hide Function
执行该元素绑定的所有隐藏函数
hide()
disable Function
设置控件不可用
disable()
enable Function
设置控件绑定
enable()
refresh Function
修改参数
/**
* @param {Object} opt 参数对象
*/
refresh(opt)
